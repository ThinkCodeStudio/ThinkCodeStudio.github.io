WT添加到右键栏

重点来了，Linux是偏向终端命令来操作文件，而在Windows环境中更偏向窗口的文件控制，但是由于WT安装时没有配置右键栏操作，所以我们就要自己配置右键栏操作。

需要操作 注册表，所以没有经验的话，必须在操作前备份注册表，如果失误就会让系统发生错误，这会很麻烦的

首先 Win + R打开运行，输入 regedit 运行就打开了注册表。

  • 文件->导出->导出范围->全部->保存

这样就备份了注册表。

ctrl + F 查找 Directory ,查看只勾选 项 ，勾选下面的 全字匹配 查找。

在 shell 项中，新建一个项，命名随意，但别命名后连自己都不知道是什么。

在右边你会看到项中的信息，(默认) 就是右键栏显示的名字，右键项新建一个 字符串值 ，命名为 Icon ,字面意思图标，双击在数据里面填写图标文件（.ico）文件的路径，但是这个值可以填 exe 文件，有兴趣试试填 png ，记住我的图标文件为 32X32 ，其他分辨率没试过。

名称 类型 数据
(默认) REG_SZ Windows Terminal
